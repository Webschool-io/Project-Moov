# Moov

> Stream Movies and tv shows

![Using](use.gif "Using")

## Requirements

- [VLC Player][2]

## Install

You can install this:


```
$ npm i -g moov
```

or just:

```
$ git clone git@github.com:uselessdev/moov.git movies
$ cd movies
$ npm install
$ npm link
$ moov
```

## Usage:

It's simple

```
$ moov search <movie-name> [options]
```

to see options available type: `moov --help`
