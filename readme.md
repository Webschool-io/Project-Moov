# Moov

> WOLOLO...

## Requirements

- [VLC Player][2]
- [Peerflix][1] installed globally

## Install

You can install this:


```
$ npm i -g moov
```

or just:

```
$ git clone git@github.com:uselessdev/moov.git movies
$ cd movies
$ npm link
$ moov
```

## Usage:

It's simple

```
& moov search <movie-name> -q [quality]
```
