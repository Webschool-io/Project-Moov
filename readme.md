# Moov

> WOLOLO...

## Requirements

- [VLC Player][2]
- [Peerflix][1] installed globally

## Install

You can clone this:


```
$ npm i -g moov
```

or 

```
$ git clone git@github.com:uselessdev/movies-cli.git movies
& cd movies
& npm link
& moov
```

## Usage:

It's simple

```
& moov search <movie-name> -r [resolution]
```
